Example: Find the exact solution to the equation: \[100(1+e^{{4w}})=1000\]

Solution

Whenever we are trying to solve an equation using the definition of logarithm, we must isolate the base entirely before we can do so. Recall that the definition is: \[ \log_B(x)=y\leftrightarrow B^y=x \] So, if we isolate the base, we can rewrite using logarithms. Here are my algebraic steps to do so: \[ \solve{ 100(1+e^{4w})&=&1000\\ 1+e^{4w}&=&10\\ e^{4w}&=&9 }\] Now we can apply the definition of logarithm (using natural log since the base is \(e\)) to get: \[ \solve{ 4w&=&\ln(9) 4w&=&\ln(3^2)\\ 4w&=&2\ln(3)\\ w&=&\frac{2\ln(3)}{4}\\ w&=&\frac{1}{2}\ln(3) } \] Notice that I take advantage of the properties of logarithms and simplify the logarthim to get \(2\ln(3)\). When possible, you should simplify powers inside logarithms. Then, when we divide both sides by four, we end up with a coefficient of one-half. There is an option to write this as \(\ln(\sqrt{3})\), but I believe the way I origianl wrote it is the most clear and simply answer.
