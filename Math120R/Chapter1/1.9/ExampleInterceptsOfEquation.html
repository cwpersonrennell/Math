Find all the \(x\) and \(y\) intercepts of the graphs for the equation:

\[ x^4 + y^3 -xy^2 = 4096 \]

Solution

What does it mean for a graph to have intercepts? There are two types of intercepts:
Vertical Intercepts: This is when the graph intercepts the vertical axis, which only occurs when the horizontal values are exactly zero. Vertical intercepts will always take the form \((0,y_1),\;(0,y_2),...(0,y_n)\)
Horizontal Intercepts: This is when the graph intercepts the horizontal axis, which only occurs when the vertical values are exactly zero. Horizontal intercepts will always take the form \((x_1,0),\;(x_2,0),\;...(x_m,0)\)

Algebraic Approach:To find the intercepts, we set the \(x\) or \(y\) value to zero, according to the definitions above.

Vertical Intercept(s):\[ \begin{array}{rcl} 0^4+y^3 - (0)y^2 &=& 4096\\ y^3 &=&4096\\ y&=&\sqrt[3]{4096}\\ y&=&16 \end{array} \] From this calcluation, we can say the there is only one vertical intercept at \((0, 16)\)

Horizontal Intercept(s): \[ \solve{ x^4 +(0)^3 - x(0) &=&4096\\ x^4 &=&4096\\ x &=&\pm \sqrt[4]{4096}\\ x&=&\pm8 } \]From this calculation, we can deduce two horizontal intercepts: \((-8,0),\;(8,0)\)
