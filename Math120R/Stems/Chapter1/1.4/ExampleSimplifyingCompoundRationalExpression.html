Example: Simplify the rational expression: \[ \dfrac{ 1+\frac{ 10 }{ x-3 } }{ 1-\frac{ 10 }{ x-3 } } \]

\[ \solve{ \dfrac{ 1+\frac{ 10 }{ x-3 } }{ 1-\frac{ 10 }{ x-3 } }&=&\dfrac{ \left(1+\frac{ 10 }{ x-3 } \right)}{\left( 1-\frac{ 10 }{ x-3 }\right) }\times{\color{ red }\dfrac{ x - 3 }{ x-3 } }\\ &=&\dfrac{ x-3+10 }{ x-3 - 10 }\\ &=&\dfrac{ x+7 }{ x- 13 } } \]
